# 命运牌局 / Fatebound Gambit Demo

**版本：v1.0 开发版**

《命运牌局》新手流程竖切片 Demo，可在手机浏览器竖屏完整体验一条从青崖峰逃亡到基地预览的主线。

稳定备份见同级目录 `FateboundGambit_Demo_v1.0_stable`（2026-06-18 冻结，只读）。**日常改动只在本目录进行，勿改稳定版。**

## 快速开始

直接打开：

```text
index.html
```

或部署到 GitHub Pages（仓库根目录包含 `index.html` 即可访问）。

**建议：** 使用手机浏览器竖屏体验；在电脑上查看时，请打开浏览器的移动端模拟器。

## 体验内容

| 阶段 | 内容 |
|------|------|
| 开篇 | 标题翻牌、青崖峰剧情、输入道号 |
| 探索 | 命运牌阵走格子、剑符移动、事件 / 灵藏 / 坊市 |
| 教学战 | 小怪 2 回合（牌型、遗物、技能） |
| Boss 战 | 3 回合传承 Combo、金鹏之心改牌 |
| 剧情杀 | 黑袍首领降临、万象传承、四条 A → 五条 A |
| 收尾 | 撤离事件、青崖峰基地预览 |

完整流程约 **5–8 分钟**。

## 开发调试

正式体验入口（不带参数）：

```text
index.html
```

开发用快捷入口（跳过部分流程）：

| 参数 | 作用 |
|------|------|
| `?skip=map` | 从命运牌阵开始 |
| `?skip=minion` | 从小怪教学战开始 |
| `?skip=boss` | 从 Boss 战开始 |
| `?skip=base` | 直接打开基地页 |

示例：`index.html?skip=boss`

对外分享的链接请使用无参数入口。
