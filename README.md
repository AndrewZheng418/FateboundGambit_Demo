# 命运牌局 / Fatebound Gambit Demo

**版本：v1.0 开发版**  
稳定备份见同级目录 `FateboundGambit_Demo_v1.0_stable`（只读，2026-06-18 冻结）。

这是《命运牌局》的海选新手流程竖切片 Demo。**后续改动只改本目录，不动稳定版。**

## 运行方式

直接打开：

```text
index.html
```

或上传到 GitHub Pages。仓库根目录只要包含 `index.html` 即可访问。

## 推荐访问

- 推荐使用手机浏览器竖屏打开。
- 电脑浏览器查看时建议打开移动端模拟器。

## Demo 内容

- 标题翻牌
- 青崖峰开篇剧情
- 输入道号
- 命运牌阵走格子
- 剑符棋子移动
- 事件、灵藏、坊市
- 小怪 2 回合教学战斗
- Boss 3 回合传承 Combo
- 皇家同花顺剧情杀
- 万象复制与五条 A
- 青崖峰基地预览
